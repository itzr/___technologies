## Key
- Use a Ohmyzsh Custom Plugin ($ZSH_CUSTOM) and configure in .zshrc

- Use Positional Parameters
- $@ (all)
- $0, $1, $2, $3 (etc)
- note:
    - $0 is the function name
    - $1 is the first parameter
